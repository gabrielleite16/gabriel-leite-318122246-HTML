Respostas

1. A internet é uma enorme rede de redes, ou melhor dizendo uma infraestrutura de redes, é ela quem faz a conecção dos computadores globalmente, formando-se assim uma rede em que qualquer computador se comunica com outro desde que estejma conectados a internet. Já a Web é uma maneira na qual as informações são disseminadas pela internet, pode-se dizer que a Web é uma grande parte da internet.

2. O que acontece quando digitamos um endereço na URL do navegador não é tão simples, possuem muitas coisas por trás. Acontece que todos os computadores possuem um endereço IP, várias portas que são por onde processos e aplicações fazem a comunicação. Para poder acessar a página da Web desejada, é necessário que o computador estabeleça uma comunicação com o computador que a página solicitada está localizada. Ou seja, o computador que quer acessar a página seria o cliente, e o computador que a página está localizada seria o servidor. Para um computador estabelecer uma conexão com outro computador, é necessário que esteja conectado a rede e estar configurado em um servidor DNS, é esse servidor que faz a comunicação entre os computadores.

3. Não são consideradas aplicações Web, porque a página HTML estática utiliza a própria CPU como servidor, as informações são armazenadas no próprio hardware, por causa disso somente quem tem acesso a esse computador consegue acessar as informações da página. Na imagem representando a explicação, o Web Server seria a CPU do computador.

4. O HTML é uma linguagem de programação muito utilizada para ambientes web como sites e blogs. Mas para ambiente desktop não, porque o HTML é utilizado para o desenvolvimentos de sites na web.
